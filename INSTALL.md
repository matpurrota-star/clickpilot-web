# ClickPilot installieren

> Diese Anleitung kannst du als `README.md` ins Release-Repo `clickpilot-releases`
> legen, damit Besucher, die direkt von GitHub kommen, eine klare Anleitung sehen.

ClickPilot ist ein KI-Technik-Assistent für Windows. Du beschreibst dein PC-Problem
in Alltagssprache, die App diagnostiziert und repariert auf Wunsch — jeder Eingriff
nur mit deiner Bestätigung.

## Voraussetzungen
- Windows 10 oder 11 (64-Bit)
- Eine Internetverbindung (für die KI-Antworten)

## In 4 Schritten

1. **Herunterladen** — lade die neueste Datei `ClickPilot_…_x64-setup.exe` von der
   [Releases-Seite](https://github.com/matpurrota-star/clickpilot-releases/releases/latest) herunter.
2. **Setup starten** — Doppelklick auf die Datei, dann dem deutschsprachigen
   Installationsassistenten folgen.
3. **Windows-Hinweis bestätigen** — falls *„Der Computer wurde durch Windows geschützt"*
   erscheint:
   - Klicke auf **„Weitere Informationen"**
   - dann auf **„Trotzdem ausführen"**
4. **Loslegen** — ClickPilot öffnet sich. Lege ein kostenloses Konto an (oder hinterlege
   deinen eigenen KI-Schlüssel) und beschreibe dein erstes Problem.

## „Unbekannter Herausgeber" — ist das ein Problem?

**Nein.** ClickPilot ist ein junges Projekt und nutzt noch kein kostenpflichtiges
Signatur-Zertifikat. Windows SmartScreen zeigt den Hinweis vorsorglich bei jeder neuen,
noch wenig verbreiteten Software — er bedeutet **nicht**, dass mit der Datei etwas nicht
stimmt. Mit zunehmender Verbreitung (und sobald ein Zertifikat hinterlegt ist)
verschwindet der Hinweis.

## Updates

ClickPilot aktualisiert sich **automatisch**. Sobald eine neue Version veröffentlicht ist,
bietet die App das Update beim Start an und installiert es nach deiner Bestätigung — du
musst nichts manuell herunterladen.

## Deinstallieren

Über *Einstellungen → Apps → Installierte Apps → ClickPilot → Deinstallieren*.
Es bleiben keine Systemänderungen zurück.
